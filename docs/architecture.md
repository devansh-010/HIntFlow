# HintFlow Architecture

## Goal

Help students learn DSA instead of cheating.

## Core Features

1. Progressive hints
2. Code-aware feedback
3. Guided debugging
4. Hint-based chat
5. LeetCode integration

## Hint Levels

Level 1 - Concept Hint

Level 2 - Direction Hint

Level 3 - Near Solution

Level 4 - Complete Solution

## System Flow

LeetCode Problem
       +
Student Code
       ↓
HintFlow Backend
       ↓
LLM Analysis
       ↓
Progressive Hint
